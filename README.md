# Installation

### Clone on your local machine

```bash
git clone https://github.com/joaoreider/real-time-board.git
```

### Navigate to project directory

```bash
cd real-time-board
```

### Install dependencies

```bash
pnpm install
```

### Rename .env.template to .env

```bash
mv .env.template .env
```

### Run 

```bash
pnpm run dev
```
